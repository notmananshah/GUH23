    Blackboard log in changes

    2-factor authentication from Duo is now required by all staff and students to log in to Blackboard. There’s more information on the IT Services website: 2-factor authentication

    Turnitin - issue when instructors save/submit settings [UPDATE]

    We have received reports that some Instructors are unable to save changes to Turnitin assignment settings. This is a known issue and relates to changes to the way recent versions of the Chrome and Edge browsers work. If you are impacted by this issue please update your browser to the latest version to resolve the problem.

    Course list displays as empty?

    Some users have contacted us to report that their Blackboard course list is displaying as empty. We have identified this as a time out and caching error. Should you see an empty course list, please clear the cache, cookies, and history on your browser, or switch to a different internet browser. You should also try logging out of Blackboard, closing your browser, and then navigating back to Blackboard and logging in again. If this doesn't work, please report the problem via the IT Support Centre.
